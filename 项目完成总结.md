## 📈 技术实现

### 计算公式（完全符合标准）
```python
# 温度校正系数
K1 = (Th - T_amb) / (100 - T_amb)
K2 = (T_amb - Tc) / T_amb

# 等效内容量
Veq = V1 × K1 + (V2 × K2) / 3

# 容许耗用能源基准
E_standard = 0.081 × Veq + 0.405

# 1-5级门槛
Grade_1: E24 ≤ 0.049 × Veq + 0.243
Grade_2: E24 ≤ 0.057 × Veq + 0.284
Grade_3: E24 ≤ 0.065 × Veq + 0.324
Grade_4: E24 ≤ 0.073 × Veq + 0.365
Grade_5: E24 ≤ 0.081 × Veq + 0.405
```

### 精度要求
- ✅ K1、K2: 小数第3位
- ✅ V1、V2: 小数第1位
- ✅ Veq: 小数第3位
- ✅ E基准: 小数第3位
